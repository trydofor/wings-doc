    # 🐵 无脑使用

    直接作为parent使用，可通过OSS获得不定期发布的SNAPSHOT版本。

    <parent>
    <groupId>pro.fessional</groupId>
    <artifactId>wings</artifactId>
    <version>${wings.version}</version>
</parent>
